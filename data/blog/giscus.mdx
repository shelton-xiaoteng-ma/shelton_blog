## 配置

打开: <https://giscus.app/>, 按照说明设置公开仓库

- 创建公开仓库, 比如: `giscus-discussions`
- 安装`giscus`app到github账号或当前仓库
- 生效`Discussions`
- 在页面配置后, 将如下消息配置在网页模版中即可生效

  ```html
  <script
    src="https://giscus.app/client.js"
    data-repo="shelton-xiaoteng-ma/giscus-discussions"
    data-repo-id="R_kgDOJhPNTA"
    data-category="Announcements"
    data-category-id="DIC_kwDOJhPNTM4CWY5z"
    data-mapping="url"
    data-strict="0"
    data-reactions-enabled="1"
    data-emit-metadata="0"
    data-input-position="bottom"
    data-theme="preferred_color_scheme"
    data-lang="en"
    crossorigin="anonymous"
    async
  ></script>
  ```

- `vuepress-theme-hope`中使用

  ```ts
  # .vuepress/theme.ts
  plugins: {
    comment: {
      /**
       * Using Giscus
      */
      provider: "Giscus",
      repo: "shelton-xiaoteng-ma/giscus-discussions",
      repoId: "R_kgDOJhPNTA",
      category: "Announcements",
      categoryId: "DIC_kwDOJhPNTM4CWY5z",
    }
  ```
