# 关于我

您好！我是 Shelton，这是我的个人博客，在这里我将分享我作为一名全栈开发者的旅程。

我热衷于构建和优化应用程序，自 2017 年以来一直专注于软件开发。我专注于使用 React、Next.js、Hono、Auth.js(NextAuth.js) 和 ShadCN 等技术开发应用。

我在数据工程方面丰富的经验，包括使用 PySpark 和 AWS 数据湖解决方案（包括 S3、Glue、Athena 和 EMR）等工具, 也为我的全栈开发工作提供了更广阔的视野和更扎实的技术支持。

我的背景包括逆向工程和网络爬虫，从事 Android 应用和 JavaScript 逆向分析工作。这段经历磨练了我的问题解决和代码分析能力。我还为 Scrapy 开发了广泛的定制化功能，实现了网络爬虫任务的集中管理和细粒度监控。我精通 MySQL 等关系数据库、Redis 和 MongoDB 等非关系数据库以及包括 Kafka 在内的消息队列。我利用这些不同的数据库和工具来实现可扩展性和数据管理。

我创建这个博客的目的是记录我的技术之旅，分享我的见解，并为其他开发者和技术爱好者提供有价值的技巧和资源。

如果您觉得我的内容有价值，我鼓励您订阅我的博客，浏览我的文章，或通过留下评论加入讨论！

我很高兴能与您分享我的经验和知识。感谢您的访问！
