# 专注于全栈开发的资深程序员

我从 2017 年开始从事编程工作，目前专注于全栈开发。主要技术栈包括 React、Next.js、Hono、ShadCN 和 Auth.js（NextAuth.js）等。

早期从事逆向工程和爬虫开发，专注于 Android 应用和 JavaScript 的逆向分析，使用 Python 和 Scrapy 开发。期间对 Scrapy 进行了大量定制化开发，实现了爬虫的集中管理和多粒度监控, 同时熟练掌握 MySQL 等关系型数据库，Redis 和 MongoDB 等非关系型数据库，以及 Kafka 消息队列。这段经历锻炼了我卓越的问题解决能力、代码分析能力以及快速掌握新技术的能力，同时积累了丰富的数据采集和管理经验。

此外，我深入参与过数据分析相关工作，熟练使用 PySpark 和 AWS 数据湖解决方案（S3/Glue/Athena/EMR）。这些经验为我的全栈开发工作提供了更广阔的视野和更扎实的技术支持。
